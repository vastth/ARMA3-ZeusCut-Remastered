# 导演手册

> 不是写"发生了什么"，而是写"我怎么导演"。
> 这一层让 AI 知道什么叫"像我风格、适合直播执行的剧本"。

---

## 一、导演职责定义

导演不是"告诉玩家发生了什么"，而是在干 6 件事：

| 职责 | 说明 |
|------|------|
| 给目标 | 这一场玩家到底要争什么（救人/抢货/卧底/谈判/撤离/报复） |
| 给阻力 | 敌人、时间限制、资源匮乏、身份暴露、盟友不可信 |
| 给选择 | 至少给 2 条可行路径，让玩家感觉"这是我做的决定" |
| 给反馈 | 玩家做了什么，世界就怎么回应（电话/广播/追兵/价格上涨/盟友翻脸） |
| 给转折 | 每场至少一个意外，让剧情活起来 |
| 给余波 | 这场结束后，岛上的局势得变，否则下一场像重开新档 |

**核心结论：导演不是写死剧本，是搭一个会对玩家行为产生反应的叙事沙盘。**

## 二、OBP 上垒模型

### O — Objective 目标

目标必须具体、可验证、能失败。

| 类型 | 示例 |
|------|------|
| ❌ 坏目标 | 推进剧情 / 做点事 / 搞点节目效果 |
| ✅ 好目标 | 在 40 分钟内把阿青从押送队手里抢出来 |
| ✅ 好目标 | 让切尔西签下停火协议，或拿到他勒索证据 |
| ✅ 好目标 | 抢在山口组前找到画里的保险柜钥匙 |

### B — Barrier 阻力（四件套）

| 阻力类型 | 说明 |
|---------|------|
| 人的阻力 | 敌对势力、摇摆盟友、贪财中间人、双面卧底 |
| 资源阻力 | 钱不够、弹药少、载具坏、通讯被截断 |
| 时间阻力 | 天亮前、车队 15 分钟后离开、援军倒计时 |
| 认知阻力 | 情报不全、目标是假消息、队内有人误判 |

> 真正让人沉浸的，不是敌人多，是"他们知道必须做，但不知道怎么做最对"。

### P — Pivot 转折（5 种类型）

| 类型 | 说明 | 示例 |
|------|------|------|
| 身份转折 | 盟友其实不是盟友 | 接头人是切尔西的人 |
| 情报转折 | 原来营救对象不是被抓，是故意被抓 | — |
| 目标转折 | 原本是护送任务，后面变成灭口或截胡 | — |
| 代价转折 | 救得了人，但必须放弃油田控制权 | — |
| 阵营转折 | 第三方俄军/警方/平民武装突然介入，战场改写 | — |

**原则：转折不是"你失败了"，而是"你必须重新判断"。**

## 三、单场时间结构（3-4 小时）

| 阶段 | 时长 | 目标 | 内容要点 |
|------|------|------|---------|
| 热身段 | 20-30 分钟 | 让人进戏，不要一上来就总攻 | 复盘上集余波 · 小任务热手 · 投放本场主线钩子 |
| 主线推进 | 60-90 分钟 | 让玩家主动做判断 | 1次调查/谈判 · 1次移动/布置 · 1次中等强度冲突 |
| 高潮段 | 30-60 分钟 | 让前面所有选择结账 | 前面的线索兑现 · 压力下做取舍 · 胜利不全拿/失败不全丢 |
| 收尾段 | 20-30 分钟 | 别把剧情演死 | 结果 + 余波 + 钩子，三样必留 |

**适合的热身开场方式：**
- 收保护费时发现目标人不见了
- 修车厂拿货时收到一通紧急电话
- 广播里传来某势力声明，打破现有平衡

## 四、剧情生成公式

**公式：现状 ＋ 欲望 ＋ 阻力 ＋ 误导 ＋ 转折 ＋ 代价 ＋ 余波**

| 项目 | 示例（阿青营救线） |
|------|-----------------|
| 现状 | 阿青被切尔西控制，位置大致已知 |
| 欲望 | 玩家想把阿青救回来并稳住平民武装 |
| 阻力 | 押送队路线不明、敌方有重火力、玩家钱不够买情报 |
| 误导 | 线人给的路线是旧路线 |
| 转折 | 真正押送目标不是阿青，是龙哥；阿青已被转移 |
| 代价 | 要救龙哥必须放走切尔西副手 |
| 余波 | 平民武装信任上升，但切尔西得到预警 |

## 五、AI 协作定位

### 导演的真实约束
- 一场直播只有 3-4 小时
- 需要实时控 NPC、控节奏、控信息
- 不能准备 30 个复杂分支，现场根本顾不过来
- 需要 80% 可复用骨架 ＋ 20% 即兴修正
- 玩家会跑偏，剧情必须模块化
- 观众要每 20-30 分钟有"可感知进展"

**AI 的定位：不是替你当导演，而是当你的副球探——帮你找便宜有效的剧情资产。**

### AI 的两个常见错误
1. 只会写小说感台词，不会做可执行任务
2. 只会出点子，不懂 Zeus 实操负担

## 六、剧情已知缺点（自诊断）

| # | 缺点 | 核心问题 |
|---|------|---------|
| 1 | 信息密度过高 | 势力/地点/NPC/电话号/历史账太多，直播前台呈现超载 |
| 2 | 主线焦点漂移 | 部分桥段与核心宿命线连接不够硬 |
| 3 | 转折是"导演发牌" | 关键情报刚好出现，玩家感觉被剧情牵着鼻子走 |
| 4 | NPC弧光不足 | 多数NPC主要承担功能，缺乏"立场、代价、偏执" |
| 5 | 主角能量偏强 | 几乎所有重大线都靠小蛋识破/联络/统筹/救场 |
| 6 | 失败分支未资产化 | 精彩主要来自"赢得漂亮"，失败尚未形成叙事资产 |
