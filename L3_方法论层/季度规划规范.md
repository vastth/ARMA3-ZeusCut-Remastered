# 季度规划规范

> 每次开新一季之前必读。本文件记录了季度规划的核心方法论。

---

## 第一步：写"一句话季度宗旨"（Season Logline）

**这是开新季最重要的第一件事，优先于任何场次规划。**

季度宗旨是一句话，回答：**这一季，主角会经历什么根本性的改变或考验？**

格式参考：
```
[主角] 以为 [他的目标/信念]，
却发现 [真正的代价/威胁/悖论]。
```

### 经典案例参考

| 剧集 | 季度宗旨 |
|------|---------|
| 《绝命毒师》S1 | "一个好人决定做一件坏事——却发现'只做一次'是个谎言。" |
| 《黑道家族》S1 | "一个黑帮老大去看心理医生，因为他开始怀疑自己选错了人生。" |
| 《火线》S1 | "警察和毒贩玩的是同一个游戏，规则是同一套——输的永远是底层。" |
| 《无间道》 | "卧底的悲剧不是死亡，是你不知道自己到底是谁。" |

## 第二步：回答"季终那一刻，主角失去了什么？"

> 这个答案决定了S全季的内容。
> 每一集都是在**慢慢剥夺那个东西**。

在正式规划第一集之前，必须能填写这句话：

```
S{X} 季终结束的那一刻，小蛋失去了：_______________
```

---

## 第三步：集数规划

### 标准直播节奏

```
3场/周 × 约4周 = 12场/月
```

### 推荐：每季 10 集

```
第 1–2 集   建立期   世界观 + 季度核心冲突引入
第 3–6 集   升级期   代价逐步累积，关系网编织
第 7–9 集   危机期   压力爆发，所有伏线收口
第 10 集    季终     弧线闭合 + 留一个跨季钩子
```

**第5集 = 中点集**，是全季张力最高的单集，应该安排最重要的"不可逆事件"。

### 为什么不是12集？

12集的后1/4（第10-12集）容易变"过渡集"——故事实质已结束，但还剩3集要撑，观众感受到拖沓。

### 为什么不是8集？

按3场/周的节奏，8集不足3周，人物关系还没来得及积累就到季终，情感冲击力不足。

## 第四步：季度结构草图（10集模板）

```
集数  标题方向         戏剧功能              必须回答的问题
──────────────────────────────────────────────────────────
1     建立             主角 + 世界 + 核心冲突  这个人是谁？他想要什么？
2     第一代价         付出第一个真实代价       他愿意为目标付出多少？
3     深入             进入新阶段/新圈子        他怎么改变自己去适应？
4     关系网           最重要的人际关系建立     他在乎谁？谁在乎他？
5     中点·不可逆      全季最重要的单集         他做了一件回不了头的事
6     压力升级         外部威胁明显化           他还能控制局面吗？
7     裂缝             最重要的关系出现裂痕     他为了目标要出卖谁？
8     全面危机         所有问题同时爆发         他能同时应对几条战线？
9     抉择             季终前的三岔口           他必须在两件重要的事之间选一个
10    季终             弧线闭合+跨季钩          他失去了什么？下一季的问题是什么？
```

---

## 第五步：季终必须留的"跨季钩子"

季终结束前，必须埋下一个**新的、无法在本季解决的问题**。

格式：
```
跨季钩子：[某人/某事] 的真相/到来，意味着 [下一季的核心威胁]。
```
