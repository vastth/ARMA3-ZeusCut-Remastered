# 原始仲裁材料 — 使用说明

> ⚠️ **这是重制版的「剧情参考本」，不是已发生的事实。**

## 这是什么

原始仲裁材料记录的是原剃刀党系列后80%的完整剧情脚本（前80%记录已丢失，重制版S1正好补上这段空白）。

重制版将以此为**参考框架**，但实际剧情会根据：
- 导演的即兴创作
- 主播和粉丝的实际反应
- ARMA3 ZEUS的现实限制

…产生不同走向。**核心角色和配角只增不减**，但事件发展路径会有差异。

## 使用原则

| 用途 | 是否可用 |
|------|---------|
| 了解核心角色的性格弧和最终走向 | ✅ 参考 |
| 了解世界观中的重要势力关系 | ✅ 参考 |
| 提取可复用的场景创意和机制 | ✅ 参考 |
| 直接照搬为重制版已发生事件 | ❌ 不可以 |
| 作为L1/L2的事实依据 | ❌ 不可以，须经导演确认后才能写入L1 |
