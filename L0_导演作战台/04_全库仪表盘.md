# 📊 全库仪表盘

> 自动聚合全库状态。数据由 Dataview 实时生成，无需手动维护。
> **只读视图** — 修改请前往对应源文件。

---

## 🎬 场次全貌

```dataview
TABLE WITHOUT ID
  file.link AS "场次文件",
  标题 AS "标题",
  执行状态 AS "执行状态",
  直播日期 AS "直播日期",
  时间线位置 AS "时间线位置"
FROM "L5_场次工作区"
WHERE 场次 AND 标题 AND 执行状态
SORT 场次 ASC
```

## 🔄 同步状态检查

> 以下文件标记了「已同步」状态，绿色 = 已完成，红色 = 待处理。

```dataview
TABLE WITHOUT ID
  file.link AS "文件",
  场次 AS "场次",
  已同步L2 AS "已同步 L2",
  已同步全局伏笔总表 AS "已同步伏笔表"
FROM "L5_场次工作区"
WHERE 已同步L2 != null OR 已同步全局伏笔总表 != null
SORT 场次 ASC
```

## 📅 最近修改的文件

```dataview
TABLE WITHOUT ID
  file.link AS "文件",
  file.folder AS "层级",
  file.mtime AS "最后修改"

SORT file.mtime DESC
```
